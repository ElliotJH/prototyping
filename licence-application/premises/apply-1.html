Apply for a premises licence
from London Borough of Westminster

Step 1 of 4
  1. Submit application
  2. Email address
  3. Pay licence fee
  4. Done

Step 1: Submit your application

Follow the instructions below:

  1. Download the application form

    Right-click the link below and select 'Save to...'

    Download the application form (PDF)

  2. Fill in the application form on your computer

    You'll need the latest version of Adobe Reader.

    Remember to save the PDF when you've finished.

  3. Submit the application form and any supporting documents

    If you can't submit everything right now don't worry, the last page of the application form contains instructions to get you back to this screen.

    Application form

    Supporting documents

    We accept the following document formats:
    PDF, Word, TXT, RTF, JPG and PNG.

    If you don't want to submit these documents electronically you can post them to the address given on the final screen of this service.
