Apply for a licence
from London Borough of Westminster

Follow the instructions below:

  1. Download the application form

    Right-click the link below and select 'Save to...'

    Download the application form (PDF)

  2. Fill in the application form on your computer

    You'll need the latest version of Adobe Reader.

    Remember to save the PDF when you've finished.

  3. Submit the application form and pay the fee

Get started

What you need to know

  • There is a £500 licence fee
  • You'll need a debit or credit card

  • You'll need to supply the following documentation:

    • Premises plan (the plans must be to scale and be clearly display the required information)
    • Consent form of premises supervisor (only required if the application includes the sale of alcohol)
