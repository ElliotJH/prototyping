Here's a proposal to reduce the number of greys we use down to four.

I've used Alex's greys as the starting point and then tweaked them slightly to make the spread more even, and made sure we've got a grey we can use for text without breaking WCAG compliance.

Hex & Sass

  • #0B0C0C, $black

  • #6F777B, $grey-1

  • #BFC1C3, $grey-2

  • #DEE0E2, $grey-3

  • #F8F8F8, $grey-4

  • #FFFFFF, $white

Semantic Sass

  • $text-colour

  • $secondary-text-colour

  • $border-colour

  • $panel-colour

  • $canvas-colour, $highlight-colour

  • $page-colour

Usage

  • Primary text

  • Secondary text

  • Lines and borders

  • Panels

  • Canvas, highlighted content

  • Page
