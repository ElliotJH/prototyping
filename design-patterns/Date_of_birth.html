Guidance

 

As with all data-gathering widgets, only use this if it's essential to the service. A person's date-of-birth is very personal information. For example, if all you want to check is that the user is old enough to use the service, use a checkbox or warning message (e.g. "By clicking submit I confirm that I am over 18").

 

If you need date-of-birth AND your service is age restricted you should reiterate this in the help text for the widget. You can also, if you want, start the year field at the earliest valid year.

 

Please avoid free text fields or calendar controls for collecting dates of birth.
