Guidance

 

As with all patterns that gather personal data, only use this if it's absolutely essential. For example, if all you want to check is that the user is old enough to use the service, use a checkbox or warning message (e.g. "By clicking submit I confirm that I am over 18").

 

If you need to collect date of birth and your service is age restricted then you should mention this in the help text for the pattern. You can also, if you want, start the year field at the earliest valid year.

 

Please avoid using free text fields or calendar controls for collecting dates of birth.
