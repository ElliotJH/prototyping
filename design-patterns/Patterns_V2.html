About this document

 

This document contains examples of some of the most common interface elements used on GOV.UK products, together with guidance on how best to use them.

 

It's been automatically generated from the prototyping package Axure. It doesn't use production-level code and is best thought of as something a bit like a PDF that you happen to view in a browser. All the production code is stored on Github and will be linked to from individual patterns, where available.

 

If you're creating Axure prototypes you can download the Axure file for this document (.rp 113k)
